WEB TECHS

HTML
HTML is a markup language that is used to create web pages. It stands for Hypertext Markup Language. HTML is used to structure the content of a web page, including text, images, links, and other multimedia elements. HTML uses tags to define the different elements of a web page, such as headings, paragraphs, lists, and tables.
CSS
CSS is a style sheet language that is used to describe the presentation of a document written in HTML. It stands for Cascading Style Sheets. CSS is used to control the layout and appearance of web pages, including colors, fonts, spacing, and positioning.
JavaScript
JavaScript is a programming language that is used to create interactive and dynamic web pages. It is a client-side scripting language that runs in the user's web browser. JavaScript is used to add functionality to web pages, such as form validation, animations, and interactive user interfaces.
